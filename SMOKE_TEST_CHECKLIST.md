# Smoke Test Checklist

Manual checklist to run after major Codex changes or before showing reports to deans/VPAA.

## 1. Basic App Load

- [ ] App opens.
- [ ] No console errors appear.
- [ ] Term tabs load.
- [ ] Main schedule grid loads.

## 2. CSV Upload

- [ ] Upload schedule CSV.
- [ ] Term appears.
- [ ] Sections display.
- [ ] Online/blank rooms are excluded from room views.

## 3. Room Tools

- [ ] Room Availability works.
- [ ] Heatmap works.
- [ ] Course Duration Graph works.
- [ ] Room Utilization works.
- [ ] PDF export works.

## 4. Enrollment Management Access

- [ ] Enrollment Management unlock works.
- [ ] Dashboard opens.
- [ ] Demand Forecast opens.
- [ ] Attrition / Lifecycle opens.
- [ ] Consolidation opens.
- [ ] Rotation section appears.

## 5. Dashboard

- [ ] Division filter changes dashboard results.
- [ ] Growth Opportunities displays.
- [ ] Reduction Opportunities displays.
- [ ] Student Presence excludes online.
- [ ] Rotation table populates.
- [ ] Dashboard summary export works.

## 6. Demand Forecast

- [ ] Historical files load.
- [ ] Forecast target can be selected.
- [ ] FTES cap fields work.
- [ ] Export CSV works.
- [ ] Export Excel works.

## 7. Attrition / Lifecycle

- [ ] Existing census-to-final calculations still work.
- [ ] Missing lifecycle milestone fields show N/A.
- [ ] Division filter works.
- [ ] Export works.

## 8. Consolidation

- [ ] Online Reduction rows appear separately.
- [ ] In-Person/Hybrid consolidation rows appear separately.
- [ ] TBA confidence capped.
- [ ] Duplicate pairwise recommendations are not shown.
- [ ] Division filter works.
- [ ] Export works.

## 9. Terminology

- [ ] User-facing text uses "Part-Time Faculty".
- [ ] No user-facing legacy part-time faculty terminology remains.

## 10. Pre-Demo Review

- [ ] Load known-good data.
- [ ] Compare 5 known courses manually.
- [ ] Export dashboard summary.
- [ ] Export detailed report.
- [ ] Confirm methodology text is visible.
